# VoltMarket PC

Интернет-магазин ПК-комплектующих на чистом HTML, CSS и JavaScript.

## Возможности

- Каталог из 44 позиций: процессоры, видеокарты, платы, память, SSD, блоки питания, корпуса и охлаждение.
- Сборщик ПК с пресетами, визуализацией системного блока и расчётом стоимости.
- Проверка совместимости: сокет CPU/платы, тип памяти, форм-фактор, длина видеокарты, запас БП и охлаждение.
- Фильтры каталога, поиск, сортировка, избранное и подбор только совместимых комплектующих.
- Корзина, промокод `BUILD7`, локальное оформление заявки и сохранение состояния в `localStorage`.
- Все ассеты лежат локально, внешние CDN и бэкенд не используются.
- Реальные фотографии комплектующих из Wikimedia Commons, источники указаны в `assets/ATTRIBUTION.md`.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8080
```

## Публикация

Сайт публикуется из ветки `main`, папка `/`.
