# VoltMarket Electronics Store

Статический интернет-магазин электроники на чистом HTML, CSS и JavaScript.

## Возможности

- Адаптивная витрина товаров.
- Поиск, категории, сортировка, фильтр по цене и наличию.
- Корзина с изменением количества и промокодом `VOLT10`.
- Избранное и сравнение товаров.
- Локальные PNG-изображения товаров, без внешних CDN и бэкенда.

## Запуск

Откройте `index.html` в браузере или запустите любой статический сервер:

```bash
python -m http.server 8080
```
